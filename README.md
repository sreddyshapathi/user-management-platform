# user-management-platform# User Management Platform

## Overview

User Management Platform is a cloud-native full-stack application demonstrating modern application development and deployment practices.

The application includes:

* User Registration
* User Login
* Profile Management
* PostgreSQL Database
* Docker Containers
* Kubernetes Deployments
* Helm Charts
* AKS Deployment
